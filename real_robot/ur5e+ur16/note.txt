We need to install librealsense from the source in the GPU laptop as it uses ubuntu 20.0

```

## git clone source, then do the fllow

sudo make uninstall   # if any previous install exists
sudo make clean
cmake .. -DBUILD_EXAMPLES=true -DBUILD_GRAPHICAL_EXAMPLES=true
make -j$(nproc)
sudo make install
sudo ldconfig

sudo cp config/99-realsense-libusb.rules /etc/udev/rules.d/
sudo udevadm control --reload-rules
sudo udevadm trigger


```
